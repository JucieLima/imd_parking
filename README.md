# PROJETO DA SEGUNDA UNIDADE
Desenvolver uma aplicação em JAVA usando a IDE Eclipse (recomendação) onde seja possível
aplicar os conceitos abaixo:

1. Classes e Objetos;
2. Herança;
3. Classes Abstratas;
4. Interfaces;
5. Composição;
6. Polimorfismo;
7. Tratamento de Exceção;
8. Coleções;
